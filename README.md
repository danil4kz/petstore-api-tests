Тестовое задание для вакансии стажера "Инженер по автоматизации тестирования  в Финтех"

Стек
- Python 3.12
- Pytest
- Requests

Запуск тестов
1. Установите зависимости:
   pip install -r requirements.txt
2. Запустите тесты:
pytest

Структура проекта
- api/pet_api.py - клиент для работы с PetStore API
- tests/test_pet.py - тесты
