Тестовое задание для вакансии стажера "Инженер по автоматизации тестирования  в Финтех"

Стек
- Python 3.12
- Pytest
- Requests

Запуск тестов
1. Установите зависимости:
   requirements.txt
2. Запустите тесты:
pytest

Структура проекта
- pet_api.py; store_api.py; user_api.py - клиенты для работы c API
- test_pet.py; test_store.py; test_user.py - тесты
